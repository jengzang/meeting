# 绵羊的秘密基地 · 使用指南

欢迎来到绵羊的秘密基地！这里是我们共同的足迹数据可视化网站项目，你可以通过交互地图、时间线、热力图和多维排行榜，回顾每一段到访和每一次出行。

---

## 页面导航

网站包含以下页面，点击任意页面的顶部导航栏即可切换：

| 页面 | 功能 |
|------|------|
| 🏠 基地 | 主页入口，展示导航卡片 |
| 🔥 热力图 | 日历热力图 + 月度统计图表 |
| 🗺 地图 | 交互地图，标记 + 交通路线 |
| 📅 时间线 | 按时间排列的活动时间线 |
| 🏆 排行榜 | 多维数据排名与分析 |

每个页面的右下角都有一只绵羊吉祥物陪伴你。

## 🔥 热力图

热力图以 GitHub 贡献日历的风格，展示某段时间内的活动频率和分布。

### 两种显示模式

点击左上角下拉框切换：

- **形状填充**：用形状区分不同日期——全满、上半、中间、下半。适合一眼看出每天的覆盖情况。
- **颜色深度**：用 4 级颜色深浅表示不同日期。颜色越深代表越重要的日子。

### 颜色主题

右上角下拉框提供 6 种配色方案：GitHub 绿、海洋蓝、赛博青、暖橙、玫瑰粉、紫罗兰。切换后整个页面配色即时改变。

### 图表区域

日历下方有三个统计模块：

- **每月占比柱状图**：各月已涂色天数/总天数的百分比，横条越长该月越充实。
- **总构成饼图**：所有月份加在一起，各月贡献占比的 3D 饼图。右侧图例列出每月天数和百分比。
- **每月占比饼图**：各月独立的迷你 3D 饼图，直观对比每月的充实度。

### 日历格交互

鼠标悬浮在任意日期格上，会显示该日期的填充说明。

## 🗺 交互地图

地图页面是所有到访地点和交通路线的可视化核心。

### 底图切换

左上角下拉框提供 13 种底图：天地图（矢量/卫星）、高德地图（矢量/卫星）、MapTiler 系列、ArcGIS 系列（含暗黑和自然地理风格）、OpenTopoMap 地形图等。切换即时生效，选择会被记住。

### 标记模式

控制面板中有三种标记模式按钮：

- **活动类型**：在地点上显示该处的主要活动名称（如"就餐""旅行""学校"），字体大小按到访次数或时长缩放。
- **地点名称**：显示地点名，超过 5 个字会截断。
- **彩色圆点**：纯色圆点，密集区域按次数显示数字。

### 尺寸模式

- **按次数**：标记大小反映到访次数的多少（对数缩放，避免极端值撑爆）。
- **按时长**：标记大小反映累计停留时长。
- 再次点击已选中的按钮可取消缩放，恢复统一大小。

### 自动聚合

勾选后，缩放级别较小时相邻地点会自动合并为一个标记，防止标记过于密集。点击聚合标记会展开多条记录的详情。注意：聚合时交通路线会自动隐藏（互斥）。

### 时间范围滑块

底部滑块可以拖动左右两端来筛选特定日期范围内的记录和交通路线。拖动时地图实时更新。点击轨道空白处可直接将滑块吸附到点击位置。

### 交通路线

勾选"显示交通"后，地图上出现彩色弧线（贝塞尔曲线），代表各段交通：

- **颜色**按交通方式区分（步行绿、机动车蓝、地铁紫、火车红、飞行青等）。
- **线宽**反映该 OD 对的累计通行时长——共 5 档，越宽越频繁。
- 线上有方向箭头指示通行方向。
- 同一 OD 对的不同交通方式会沿同一弧线分段展示。
- 点击任意交通线弹出详情，显示各段类型和时长。

### 热力图覆盖

勾选"热力图"后，地图上叠加一层热力图层——红色代表密集区域，蓝色代表稀疏区域。密度和半径可在配色面板中调节。

### 配色面板

点击控制面板中的"配色"按钮，弹出配置卡片：

- 可以为每种活动类型单独设置标记颜色。修改即时生效并自动保存。
- 调节热力图密度（影响热力点的权重）和半径（影响热力扩散范围）。
- 点击"重置配色"恢复默认。

### 详情弹窗

点击任意标记弹出详情面板，包含：

- 活动类型和地点名
- 精确到分钟的时间范围
- 停留时长
- 备注（如果有）
- 详细地址（含国旗、省、市、区县、街道）
- 逐时天气数据（温度、体感温度、风向风速、湿度、能见度、紫外线、降水概率和降水量）

多记录聚合标记会列出每条记录的详情，地址和天气自动去重合并。

### 控制面板

- 点击右上角 ✕ 按钮可以收起控制面板，给地图腾出更多空间。
- 收起后右上角出现"⚙ 打开面板"按钮，点击可重新打开。
- 点击"重置视角"按钮，地图自动飞回所有标记的居中视角。

### 键盘与手势

- 鼠标滚轮缩放地图
- 拖拽平移地图
- 左上角 +/- 按钮也可以缩放
- 移动端支持双指缩放和滑动

## 📅 时间线

时间线以日为列、时间为纵轴，将活动记录和交通段排列在时间轴上。

### 视图结构

- **左侧**：24 小时时间刻度（00:00 ~ 24:00）
- **顶部**：日期列头，显示"月/日"和星期几
- **主体**：每个日期列内，活动块按时间从上到下排列
- 鼠标悬浮在任意活动块上，弹出浮动提示框显示活动类型、地点和时间范围

### 紧凑模式

勾选"紧凑模式"后：

- 日列变窄，所有日期可以在一屏内看到更多天
- 活动块内隐藏文字标签，只保留色块
- 24 小时高度自动适配屏幕，不需要上下滚动
- 仍然可以鼠标悬浮查看详情

### 活动与交通

时间线同时显示到访记录和交通段：

- **活动块**：实心色块，颜色对应活动类型
- **交通块**：颜色对应交通方式，副标题显示起终点

注意：跨天的活动会自动按天拆分展示。

## 🏆 排行榜

排行榜从多个维度分析数据，提供排名和数据洞察。

顶部摘要栏显示总览：城市数、总到访次数、总时长、总地点数。

### 🏙 城市

- **默认按到访次数降序**排列所有城市。
- **点击表头**可按任意字段排序（到访次数、总时长、地点数、活动类型数）。再次点击切换升降序。
- **点击任意城市行**展开详情面板：
  - 活动分布——各类活动的横向柱状图
  - 交通方式——各交通类型的次数、总时长、平均/最长/单日最长、涉及天数
  - 天气概况——温度（最低/最高/平均）、体感温度、湿度、降水、风速、能见度、紫外线的全维度统计，以及天气类型频次和日期范围

### 📍 区县

与城市 Tab 逻辑相同，但粒度细化到区县级别。同样支持排序和展开详情。

### 🌤 天气

- 顶部切换"城市"和"区县"视角。
- 9 个排名卡片，每个展示 Top 10：

| 卡片 | 说明 |
|------|------|
| 最热 | 最高气温排名 |
| 最冷 | 最低气温排名 |
| 最高体感温度 | feels like 最高 |
| 最大风速 | 风速极值 |
| 累计最多降雨 | 总降水量 |
| 单次最大降雨 | 单小时最大降水 |
| 最低能见度 | 能见度最差 |
| 最高紫外线 | UV 指数最高 |
| 最潮湿 | 平均湿度最高 |

每个排名项显示位置名称、数值和发生时间。

### 🚗 交通

- "城市"模式：按交通类型分组，每组显示各城市的使用次数和时长统计。
- "单段"模式：展示每种交通方式的最长单次行程，包含起终点名称、时长、占 OD 总程的百分比、所在城市。

### 🌍 地理

展示活动范围的地理极值：

- 最北/最南/最东/最西到访点——精确到坐标和城市
- 最远点——距原点最远的到访位置，标注距离（公里）
- 经纬跨度——南北纬度差、东西经度差

### 🧭 旅程

- 平均行程时长
- 出发/到达 24 小时分布柱状热力图（从凌晨 5:00 起算，因为行程通常不跨越清晨）
- 极值记录：最早/最晚出发、最早/最晚到达、最长单次行程的详细信息

## 🐑 绵羊吉祥物

右下角的绵羊是基地的吉祥物，有很多互动彩蛋：

- **瞳孔跟随**：绵羊的眼睛会跟着你的鼠标移动。
- **悬浮说话**：鼠标移到绵羊身上，它会随机说一句话（"咩~""你好呀！""今天天气不错~"等）。
- **点击变装**：每次点击绵羊，它会切换到一个随机造型（包括眯眼笑、行走、睡觉、宝宝、好奇等变体），并爆发星星粒子。
- **双击爱心**：快速双击绵羊，触发爱心形爆发效果。
- **自动变装**：绵羊偶尔会自己切换造型。
- **稀有造型**：约 5% 概率出现戴蝴蝶结或礼帽的稀有绵羊。
- **页面过渡**：点击页面间的导航链接时，一只绵羊会从左侧跑过屏幕，然后进入新页面。

## 其他小功能

### 页面换肤

热力图页面支持 6 种主题。其他页面的视觉风格为统一的暗色主题（深色背景 + 玻璃态面板）。

### 数据保存

你在页面上的设置（底图选择、标记模式、配色方案等）会自动保存。下次打开页面时会自动恢复。

### 移动端

所有页面均适配移动端。时间线页面建议横屏使用以获得更好的体验。
